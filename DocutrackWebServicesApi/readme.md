This module contains DocuTrack Web Services generated classes

There are several build profiles:
- cached-sandbox-wsdl - Generate classes based on cached wsdl in resources folder. Active by default
- sandbox-wsdl - Generate classes based on wsdl hosted by DocuTrack. Note that your IP should be whitelisted by DocuTrack to access Web Services. Currently ScineceSoft developers IPs are allowed (37.17.25.0/27)
